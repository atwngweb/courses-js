A. Toán tử số học
    + (Cộng): Cộng hai giá trị.
    - (Trừ): Trừ một giá trị từ giá trị khác.
    * (Nhân): Nhân hai giá trị.
    / (Chia): Chia một giá trị cho giá trị khác.
    % (Chia lấy dư): Tính phần dư của phép chia.
    ++ (Tăng giá trị lên 1): Tăng giá trị của biến lên 1.
    -- (Giảm giá trị đi 1): Giảm giá trị của biến đi 1.
    ** (Lũy thừa): Tính lũy thừa của một số.  



B. Toán tử gán
    Toán tử gán (=): Gán giá trị cho biến.
    Toán tử gán kết hợp (+=, -=, *=, /=, %=, **=): Kết hợp phép toán với phép gán.

D. Toán tử ++i và i++

    i++ => Có nghĩa là hậu tăng => trả về giá trị hiện tại của biến RỒI MỚI TĂNG

    ++t => tiền tăng => tăng giá trị của biến trước RỒI MỚI TRẢ VỀ GIÁ TRỊ




E. Các loại hàm trong js

    1. Hàm khai báo (Declaration function)

        function tenHam(thamSo) {
            // Khối lệnh
        }

    2. Hàm biểu thức (Function expression) 
         
        const tenHam = function(thamSo) {
            // Khối lệnh
        };
    
    (*) Hàm này chỉ có thể được gọi sau khi nó được định nghĩa.

    Lưu ý: Nếu gọi hàm trước khi định nghĩa sẽ xảy ra lỗi: Uncaught ReferenceError: Cannot access 'tinhHieu' before initialization.

    3. Hàm mũi tên (Arrow function)

        const tenHam = (thamSo) => {
            // Khối lệnh
        };
    
    Hàm mũi tên: Cú pháp ngắn gọn của hàm biểu thức được giới thiệu trong ES6.

        const tinhTich = (a, b) => {
            return a * b;
        };

        const tinhTich = (a, b) => a * b;
